Passos restantes

Só falta configurar o Firebase (banco de dados) e publicar online.

1
Ativar Firestore no Firebase
Acede a console.firebase.google.com → projeto Car-Garantia
No menu: Build → Firestore Database → Create database
Escolhe "Start in test mode" → região us-east1 → Enable.
2
Copiar as credenciais do Firebase
Clica no ⚙️ → Configurações do projeto → role até "Seus apps"
Clica em "Web" (</>) → regista a app → copia o bloco firebaseConfig.
3
Colar no ficheiro index.html
Abre o ficheiro num editor de texto (Bloco de Notas funciona).
Encontra a secção 🔧 COLE AQUI AS SUAS CREDENCIAIS
Substitui os COLE_AQUI_... pelos teus valores reais.
4
Publicar no GitHub Pages
Cria conta em github.com
Cria repositório car-garantia (Public) → faz upload do index.html
Settings → Pages → Branch: main → Save
O teu link: https://SEU_USUARIO.github.io/car-garantia 🎉
